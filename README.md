<h1>Washington DC Trip Registration</h1>

<h2>Description</h2>
A website that displays the itinerary for a 3 day trip to Washington D.C. and allows visitors to register for an informational meeting as well as vote for their favorite activity.
<br />


<h2>Technologies Used</h2>

- <b>JavaScript</b>
- <b>Python (Flask, Python DB-API)</b>
- <b>HTML/CSS</b>
- <b>SQL</b>
